Para más detalles nuestro amigo @Rosmel Ortiz proporcionó el siguiente contenido

Ver versión Git:

            
              git --version

Configuración inicial Git:

            
              git config

Inicializar un proyecto:

            
              git init

Mover archivos al "Staging Area":

            
              git add

Ver estado de los archivos del proyecto.:

            
              git status

Mover archivos al "Local Repository":

            
              git commit

Ver logs de commits:

            
              git log

Agregar repositorio remoto:

            
              git remote

Subir cambios al repositorio remoto:

            
              git push

Bajar cambios del repositorio remoto:

            
              git pull
