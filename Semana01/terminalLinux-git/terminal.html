Lista el manual de cualquier comando:

            
              man

Lista el contenido de un directorio:

            
              ls

Impríme en pantalla el directorio en el que se encuentra actualmente: 

            
              pwd

Navegar entre directorios:

            
              cd

Crear archivos:

            
              touch

Editar texto desde terminal:

            
              nano

Limpiar la terminal:

            
              clear

Mostrar contenido de un archivo:

            
              cat

Crear directorios:

            
              mkdir

Navegar entre directorios:

            
              cp

Copiar elementos:

            
              cd

Mover o renombrar elementos:

            
              mv

Remover un archivo o directorio:

            
              rm
